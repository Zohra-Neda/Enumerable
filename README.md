# ğŸ“– [Enumerable] <a name="about-project"></a>

**[Enumerable]** In this project, we learn about using modules in Ruby by building our own version of a portion of the Enumerable module. We create a class called MyList that includes an instance of the MyEnumerable module. This module implements several important methods found in Enumerable, including all?, any?, and filter.

### Key Features <a name="key-features"></a>

- **[all?: Returns true if the block returns true for all elements in the collection, false otherwise.]**
- **[any?: Returns true if the block returns true for at least one element in the collection, false otherwise.]**
- **[filter: Returns a new collection containing only the elements for which the block returns true.]**
